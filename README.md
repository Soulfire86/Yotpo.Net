# Yotpo.Net
.NET client library for Yotpo Reviews Platform API

![Bulid Status](https://ci.appveyor.com/api/projects/status/k0fctg055sy8l826?svg=true "Build Status")

## Methods Implemented
- **Authentication**
  - Request OAuth Token
- **Purchases**
  - Create an Order
  - Delete a Purchase
- **Users**
  - Create a New User
  - Retrieve User Profile Data
- **Mail After Purchase**
  - Send Test Email
